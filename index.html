What is Phyth?

It's an API framework that focuses on speed of deployment and simplicity.

If you're looking to create an API quickly, most people default to php. This is because nearly every server is configured to support php by default. The problem with php is that it lacks the feature set and scalability of other languages like java or python. If you're looking to create an API in one of these more complete languages, most people will turn to a framework like Django. The problem with Django is the amount of configuration and set up required makes it useless in certain scenarios such as hackathons and proof of concepts.

Phyth does not have any external dependencies other then vanilla php and python. This allows Phyth to work on any unix bases system out of the box.

How do I use it?

To install Phyth, simply download the source and extract it to wherever you want Phyth to live on your web server. Point your browser to the Phyth directory to load the test page (phyth/index.php). If you see all green, Phyth is up and running.

Adding functions to your API is done by the creation of module. A module is simply a python file that follows a few simple conventions. See the template below to get started on your first module:

import phyth
def main():
    phyth.start(function)

def function():
    ## YOUR CODE STARTS HERE ##

    # this is how you retrieve the sent via the request
    data = phyth.getData()

    # this is where you put the functionality of the module

    # this is how you send your reply
    phyth.respond(some_dictionary)

    ## YOUR CODE ENDS HERE ##

if __name__ == "__main__":
    main()

To call an API function, aka module, you must send a post or get request to the phyth.php. Thil will typically be www.yourwebsite.com/phyth/phyth.php. Your request must contain a variable called function. This is what tells Phyth will API function is being called. the function is derived from the file's name, so to call reverse.py you would send reverse as the function. You can send in any aditional data you wish. Here is an example of a get request to a module that reverses a string: 

www.yourwebsite.com/phyth/phyth.php?function=reverse&string=johndoe

The example request above would return the following in JSON:

{"data":{"new_string":"eodnhoj"}, "error":""}

How do I set it up on my server?

1) Download the source code from github

2) Extract it to the root of your webserver

3) Done. Go create some modules!
